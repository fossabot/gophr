============
Installation
============


Stable release
--------------

To install Gophr CMS, run this command in your terminal:

.. code-block:: console

    $ pip install --upgrade gophr

This is the preferred method to install Gophr CMS, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for Gophr CMS can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/aeroxis/gophr

Or download the `tarball`_:

.. code-block:: console

    $ curl  -OL https://github.com/aeroxis/gophr/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Github repo: https://github.com/davydany/gophr
.. _tarball: https://github.com/davydany/gophr/tarball/master
